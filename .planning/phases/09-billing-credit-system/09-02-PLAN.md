- `createCheckoutSession(packId: string): Promise<{ url: string }>` -- calls `supabase.functions.invoke('create-checkout', { body: { packId } })`, returns `data as { url: string }`

  - queryFn: calls `fetchCreditBalance(profile!.advertiser_id!)`
  - enabled: `!!profile?.advertiser_id`
  - refetchInterval: `30_000` (poll every 30 seconds)
  - staleTime: `10_000` (consider stale after 10 seconds)

    5. If balance is above threshold or 0: dismiss the toast via `toast.dismiss('low-balance')`
