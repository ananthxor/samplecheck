# Plan 10-05: Reporting & Billing Parity

**Goal:** Extend the new interaction metrics (Engagement Rate, Video Completion) to the Custom Reports builder and Billing consumption tables.

## Tasks

1. **Task 1: Update Report Types & Builder (auto)**
   - Add `engagement_rate` and `video_completion_rate` to `ReportMetric` type in `report-types.ts`.
   - Update `METRIC_OPTIONS` to include these new metrics.
   - Update `ReportsPage.tsx` to aggregate and display these cards when selected in an active report.

2. **Task 2: Update Billing Consumption Table (auto)**
   - Update `CreativeConsumptionTable.tsx` to include Engagement Rate and Video Completion columns.
   - Update `use-billing-consumption.ts` hook to calculate these fields for each row.

3. **Task 3: Final Verification (auto)**
   - Ensure consistent naming and formatting (2 decimal places + %) across Analytics, Reports, and Billing.

## Verification
- Open Reports -> New Report -> Verify new metrics appear in checkboxes.
- Create a report with Engagement Rate -> Run -> Verify card appears with correct value.
- Open Billing -> Verify table has new interaction columns.
