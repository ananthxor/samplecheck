<task type="auto">
  <name>Task 3: Create campaign placements tab component (CAMP-09)</name>
  <files>apps/web/src/features/campaigns/components/campaign-placements-tab.tsx</files>
  <action>
Create a new component `CampaignPlacementsTab` that lists all campaign creatives with inline tag copy buttons.

Props interface:
```typescript
interface CampaignPlacementsTabProps {
  campaignId: string
  creatives: Tables<'creatives'>[]
}
```

Implementation:
1. Import `generateDfpTag`, `generateEmbedTag`, `getServeBaseUrl` from `../lib/tag-generator`.
2. Import `Tables` from `@scrolltoday/shared`.
3. Import `Copy`, `Check` from `lucide-react`, `toast` from `sonner`, `Button` from `@/components/ui/button`, `Badge` from `@/components/ui/badge`.
4. Add state: `const [copied, setCopied] = useState<string | null>(null)` to track which button was just clicked.
5. Create a `handleCopy` async function matching the pattern in `tag-export-dialog.tsx`:
   ```typescript
   async function handleCopy(text: string, key: string) {
     try {
       await navigator.clipboard.writeText(text)
       setCopied(key)
       setTimeout(() => setCopied(null), 2000)
     } catch {
       toast.error('Failed to copy to clipboard')
     }
   }
   ```
6. Get `serveBaseUrl` once at the top of the component: `const serveBaseUrl = getServeBaseUrl()`.
7. Empty state: if `creatives.length === 0`, show a centered message: "No creatives assigned to this campaign yet."
8. Render a responsive table using standard HTML `<table>` with Tailwind classes (matching the project's table patterns). Columns:
   - **Creative** -- name (truncated) + format name and dimensions below in muted text
   - **Status** -- use `StatusBadge` component from `../components/status-badge`
   - **DFP/GAM Tag** -- Copy button
   - **Embed Code** -- Copy button
9. For each creative row:
   - Generate tags: `const dfpTag = generateDfpTag({ creativeId: creative.id, width: creative.width ?? 300, height: creative.height ?? 250, serveBaseUrl })`
   - Same for embedTag.
   - Determine if tags are copyable: `const canCopy = creative.status === 'active' || creative.status === 'paused'` (Research Pitfall 4: disable for draft/archived).
   - DFP copy button: `<Button variant="outline" size="sm" disabled={!canCopy} onClick={() => handleCopy(dfpTag, creative.id + '-dfp')}>`. Show Check icon + "Copied!" when `copied === creative.id + '-dfp'`, otherwise Copy icon + "DFP".
   - Embed copy button: Same pattern with key `creative.id + '-embed'` and text "Embed".
   - If `!canCopy`, add a `title` attribute on the button: "Activate this creative to enable tag export".
10. Wrap the table in `<div className="overflow-x-auto rounded-lg border">`.
11. Add a header above the table: `<p className="text-sm text-muted-foreground mb-4">{creatives.length} creative(s) assigned. Copy ad tags for active or paused creatives.</p>`

Export as named export: `export function CampaignPlacementsTab(...)`.
  </action>
